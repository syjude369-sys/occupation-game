DD3 Lab Pattern Engine Furniture Layout Interface Guide

Pattern Engine Interface Guide

이 페이지는 Pattern Engine의 오른쪽/왼쪽 제어 패널과 archive, motif, furniture overlay 흐름을 발표용으로 설명하기 위한 요약 문서입니다.

Discovery

Interval turns
시뮬레이션 몇 턴마다 패턴 후보를 판독할지 정합니다.
Min occupied cells
패턴으로 인정할 최소 점유 칸 수입니다. 현재 기본값은 3칸입니다.
Min occupancy
패턴 내부 칸 중 점유된 칸의 최소 비율입니다. 1/3 미만이면 버립니다.
Max occupancy
너무 꽉 찬 패턴을 거를 때 쓰는 최대 점유 비율입니다.
Max region area
region-based discovery가 검사할 최대 영역 크기입니다.
Sliding Window
고정 크기 창으로 보드를 훑으며 반복되는 국소 패턴을 찾습니다.
Region-Based
연결된 점유 영역을 기준으로 패턴 후보를 추출합니다.

Pattern Fitness

Recurrence
여러 test, seed, turn에서 반복해서 등장하는 정도입니다.
Distinctiveness
다른 흔한 형태와 구분되는 고유성입니다.
Symmetry
회전/반사 구조가 갖는 질서도입니다.
Void structure
패턴 안의 빈 공간이 연결성, 출입 가능성, 내부성 측면에서 갖는 점수입니다.

Objectives

Continuity
각 플레이어 영역이 하나의 덩어리로 유지되는 정도입니다.
Interpenetration
서로 다른 영역이 얼마나 얽혀 있는지 봅니다.
Boundary
서로 다른 점유 영역 사이의 접촉 경계량입니다.
Balance
플레이어별 점유량 균형입니다.
Stability
반복 실행에서 유사한 최종 형태가 나오는 정도입니다.
Active motif
등록한 motif fitness를 켠 뒤에만 최적화 목적에 포함됩니다.

Project Data

Export JSON
현재 archive, motifs, discovery 설정을 파일로 저장합니다.
Import JSON
저장한 프로젝트 상태를 다시 불러옵니다.
Reset saved data
브라우저 localStorage에 저장된 현재 프로젝트를 초기화합니다.

기본 Workflow

1. Initial pattern searchmotif fitness가 꺼진 상태에서 무작위 시뮬레이션을 돌려 pattern 후보를 수집합니다.
2. Promote motifDiscovered 목록에서 의미 있는 패턴을 motif로 등록합니다.
3. Run generationmotif fitness를 활성화한 뒤, 등록된 motif가 더 잘 유지되도록 generation을 돌립니다.
4. Furniture overlaymotif를 Furniture Lab으로 보내 layout을 선택하면 보드 위 실제 발생 위치에 도면기호가 표시됩니다.

Archive와 Replay

Archive는 각 candidate run의 결과를 보관합니다. 카드의 `View`는 보드 상태를 표시하고, `Replay`는 turn 0부터 마지막 turn까지 발전 과정을 재생합니다. Replay bar의 `Save video`를 누르면 `.webm` 파일이 브라우저 다운로드 폴더에 저장됩니다.

Motif hits는 단순 등장 횟수가 아니라, 최소 3턴 이상 존속한 motif episode를 기준으로 집계합니다.

Motif Overlay

Motif overlay는 preview 위치가 아니라 현재 보드에서 실제로 발견된 occurrence 위에 표시됩니다. overlay 체크가 켜져 있으면 motif marking과 furniture layout marking이 함께 유지됩니다.

Furniture symbolFurniture Lab에서 선택한 layout입니다. 보드 위 motif 발생 위치에 CAD 도면기호로 표시됩니다.
Player area각 플레이어가 점유한 셀입니다. 색상은 플레이어 index를 나타냅니다.
Occupied group시뮬레이션 규칙에 따라 확장된 점유 영역입니다.

Furniture Lab 연결

Motif 카드의 `→ Furniture` 버튼은 해당 motif의 크기, cell 구조, archive source를 Furniture Lab URL로 전달합니다. Furniture Lab에서 layout을 생성하고 `Apply to Pattern Board`를 누르면 선택된 layout이 별도 localStorage layer에 저장됩니다. Pattern Engine은 이 저장값을 읽어 motif id 또는 signature가 맞는 경우에만 overlay를 그립니다.
